DESCRIPTION DU PROJET

Le projet consiste à réaliser en langage C
un jeu inspiré du célèbre jeu :
« LE MOT LE PLUS LONG ».

Le principe est de générer une grille
de 9 lettres à partir des choix des joueurs.

Chaque joueur propose ensuite un mot.

Le programme :
- vérifie si le mot peut être formé
  avec les lettres de la grille,
- vérifie si le mot existe dans le
  dictionnaire,
- attribue automatiquement les points,
- affiche les scores,
- détermine le gagnant final.
